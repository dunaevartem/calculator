Данное веб-приложение написанное на языке golang с помощью написанного пайплайна _gitlab-ci.yml_ билдится, тестируется (_main_test.go_ если добавить в папку _test_ другие тесты то они поочередно будут выполняться) и деплоится на физ./вирт. сервер в виде службы через написанный ansible-playbook (_deploy.yml_).\
\
Для корректной отработки пайплайна необходимо осуществить ssh-соединение между раннером и сервером для деплоя, этот ssh-ключ будет скопирован и отправлен с раннера в созданный им контейнер докера.\
\
В файле _hosts_ можно перечислить все серверы на которых требуется установить данное веб приложение.\
\
Файл _go.mod_ — это описание проекта (какие зависимости нужно использовать, их версии, и какую версию использует Gin). Файл _go.sum_ — это хеш-суммы зависимостей (чтобы проверить целостность файлов и предотвратить подделку, хранит хеш-суммы для Gin).\
\
Файл _myapp.service_ создан для поднятия веб-приложения в виде службы.
