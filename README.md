Данное веб-приложение написанное на языке golang с помощью написанного пайплайна __gitlab-ci.yml__ билдится, тестируется (__main_test.go__ если добавить в папку __test__ другие тесты то они поочередно будут выполняться) и деплоится на физ./вирт. сервер в виде службы через написанный ansible-playbook (__deploy.yml__).\
\
Для корректной отработки пайплайна необходимо осуществить ssh-соединение между раннером и сервером для деплоя, этот ssh-ключ будет скопирован и отправлен с раннера в созданный им контейнер докера. В гитлабе необходимо задать переменную SSH_PRIVATE_KEY\
\
В файле __hosts__ можно перечислить все серверы на которых требуется установить данное веб приложение.\
\
Файл __go.mod__ — это описание проекта (какие зависимости нужно использовать, их версии, и какую версию использует Gin). Файл __go.sum__ — это хеш-суммы зависимостей (чтобы проверить целостность файлов и предотвратить подделку, хранит хеш-суммы для Gin).\
\
Файл __myapp.service__ создан для поднятия веб-приложения в виде службы.
